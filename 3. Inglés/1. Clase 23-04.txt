A good software developer writes clean and efficient code. They use powerful tools like text editors, debuggers, and
version control systems. The developer tests the program carefully to find and fix errors. A fast computer helps them
work better. Some words in programming, like “actual,” look familiar but have different meanings in English. Learning
technical terms and grammar improves communication in the tech world.

Un buen desarrollador de software escribe código de manera eficiente y limpia. Usan potentes herramientas como 
editores de texto, depuradores y sistemas de control de versiones. El desarrollador pone a prueba el programa de
manera detallada para encontrar y solucionar errores. Una computadora rápida los ayuda a trabajar mejor. Algunas
palabras en la programación, como "actual", lucen familiares pero tienen diversos significados en ingles. Aprender 
términos técnicos y la gramática mejora la comunicación en el area de tecnología.


1. Yo he accedido al sistema sin problemas. (Presente perfecto)

	I have accessed the system

2. Tu has añadido muchos archivos al proyecto. (Presente perfecto)

	You have added a lot of files to the project

3. El había descargado el archivo antes del error. (Pasado perfecto)

	He had downloaded the file before the error / bug

4. Nosotros habíamos configurado todo correctamente. (Pasado perfecto)

	We have configured everything right.

5. Ellos habrán instalado el software para el lunes. (Futuro perfecto)

	They will have installed the software by Monday.

6. El técnico habrá corregido los errores para el viernes. (Futuro perfecto)

	The technician will have fixed the bugs by Friday

7. Ella ha creado tres usuarios nuevos hoy. (Presente perfecto)

	She has created three new user's today

8. Nosotros habíamos eliminado el virus antes del escaneo. (Pasado perfecto)

	We have removed the virus before the scan

9. El sistema habrá compilado el código para entonces. (Futuro perfecto)

	The system will compile the code by then.

10. He leído el informe del sistema esta mañana.  (Presente perfecto)

	I have read the system report this morning.
